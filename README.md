## 🛠️ How to Use LiteDoc

1. Open the LiteDoc app using the desktop icon.
2. Click the button labeled Open PDF.
3. Select the file you want to convert.
4. Click the Convert button.
5. Wait for the progress bar to finish.

The software places your Markdown file in the same folder as your original PDF. The file keeps the same name but uses a .md ending. You can open this file with Notepad, WordPad, or any Markdown editor.

## ❓ Frequently Asked Questions

**Does LiteDoc work without the internet?**
Yes. You can disconnect your internet and the app still functions. All parts of the conversion happen on your local hardware.

**Are my files saved on a cloud server?**
No. This app lacks any cloud features. Files remain on your local hard drive unless you move them yourself.

**Why does it use a browser window?**
Browser technology allows for a clean and fast user interface. It makes the app easy to use. It does not mean the app is an online service.

**What happens if the conversion fails?**
Check if your PDF has a password. The app cannot convert locked or encrypted files. Ensure the file is not corrupted. If the problem persists, try to open the PDF in a reader and print it to a new PDF file before using LiteDoc.

**Can I change where the files go?**
Yes. Look for the Settings menu in the app. You can change the default output folder to any location on your computer.

**Is this software free?**
Yes. You use the software without any subscription or payment.

## 💡 Tips for Better Results
